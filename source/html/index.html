https://api.beethovens-werkstatt.de

Welcome to the public API of the Beethovens Werkstatt project. You can find more information on that project at https://beethovens-werkstatt.de.

This domain is intended as an API endpoint, there is no website content available over here. A good starting point to use this API is the following endpoint: https://api.beethovens-werkstatt.de/iiif/documents.json. A more complete OpenAPI documentation of this API will be added at a later stage of the project. While we're aiming at stable references for use in a Linked Data context, we cannot ensure persistent identifiers (or content) at this point yet, so please consider all contents as temporary for now.

If you have any questions, please get in touch with us via the corresponding GitHub repository.
